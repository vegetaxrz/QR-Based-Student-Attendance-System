Important: Wokwi Network Limitation

Why can't we connect to Wokwi ESP32?

Wokwi runs ESP32 in a virtual network (IP: 10.10.0.2) that your browser cannot access directly. This is a security limitation of web browsers - they can't reach simulated networks.

Solutions:

1. Demo Mode (Current): This page simulates ESP32 validation locally in JavaScript

2. Real Hardware: Upload code to physical ESP32-S2 on your WiFi network

3. Wokwi Bridge: Use Wokwi's paid API bridge feature (requires subscription)

Demo Mode is active! The system works exactly like ESP32 would - it validates attendance, prevents duplicates, and tracks everything. Only difference: validation happens in browser instead of ESP32.
